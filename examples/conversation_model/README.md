# Simple Conversation Model

A beginner-friendly tutorial demonstrating how to create LLM-powered agents that interact with each other.

## What This Example Demonstrates

- Creating agents with different personalities
- LLM-powered conversation generation
- Agent-to-agent communication
- Basic data collection

## Running the Example

1. Install mesa-llm:
```bash
pip install mesa-llm
```

2. Set your API key:
```bash
export OPENAI_API_KEY="your-key-here"
```

3. Run the model:
```bash
python conversation_model.py
```
